# pyBaCy

This is a python package to coupled numerical weather prediction models to
 data assimilation algorithms.
 This package was used in the following papers and will be used in further
  papers about data assimilation:
   - Finn, T. S., Geppert, G., & Ament, F. (2020). Towards assimilation of wind profile observations in the atmospheric boundary layer with a sub-kilometre-scale ensemble data assimilation system.
